## About

I'm a software engineer and researcher based in Nigeria, focused on building reliable systems and contributing meaningfully to the open source ecosystem. I work primarily in Python and C, with a strong background in backend infrastructure, systems programming, and embedded development.

Beyond writing code, I care about communicating ideas clearly — whether through technical documentation, blog posts, or open source contributions that others can actually learn from.

- 🔭 Currently deepening my expertise in systems design and distributed architectures
- 📝 I write about software engineering on [Medium](https://medium.com/@dilibe) — topics ranging from networking internals to open source culture
- 🌍 Ranked among top GitHub contributors in Nigeria
- 📫 Reach me at **franklinfidelugwuowo@gmail.com**
